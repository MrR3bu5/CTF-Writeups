## Entry format I use

For each target I include:
- Metadata: OS, difficulty, status, date completed
- Recon: what mattered, proof artifacts
- Initial access: the pivot and proof
- Privilege escalation: root cause, proof
- Defensive takeaways: fix and detection ideas
