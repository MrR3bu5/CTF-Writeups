# CTF Writeups

This repo documents CTF style labs I have completed.
I focus on methodology, offensive reasoning, and defensive takeaways.

I do not publish flags, passwords, private keys, tokens, or copy and paste exploit strings.
I only publish full solutions when the platform rules allow it, for example Hack The Box retired content only. [web:17]

## Repo structure

CTF-Writeups/
- TEMPLATE/
- HackTheBox/
- TryHackMe/

## Platforms

- HackTheBox: `HackTheBox/README.md`
- TryHackMe: `TryHackMe/README.md`

## What you will find in each writeup

- Target info, OS, difficulty, status, date completed
- Recon notes, what I checked, what mattered
- Initial access, the pivot and proof
- Privilege escalation, the path and root cause
- Defensive notes, fix and detection ideas

## Sanitization rules

- No flags.
- No credentials.
- No private keys.
- No active target spoilers.
- Redact sensitive hostnames and tokens.

If you see something that should be removed, open an issue.
